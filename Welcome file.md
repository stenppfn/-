## 5. 运算符

算数运算符： `+  - * /`

| 算术运算 | 含义 | 举例 | 运算结果 |
| -------- | ---- | ---- | -------- |
| `+`      | 加法 | 3+5  | 8        |
| -        | 减法 | 10-5 | 5        |
| *        | 乘法 | 2*3  | 6        |
| /        | 整除 | 10/3 | 3        |
| %        | 取值 | 10%3 | 1        |
